Automated Web Application Deployment using AWS EC2, Docker and Jenkins

Continuous Integration & Continuous Deployment

This project demonstrates an end-to-end DevOps workflow where code changes pushed to GitHub are automatically built, containerized using Docker, and deployed on an AWS EC2 instance through Jenkins automation.
